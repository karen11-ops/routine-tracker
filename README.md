
🌸 ルーティン記録アプリ
毎日のルーティンをチェックしてポイントを貯め、月末のお小遣いに換算するための家族用Webアプリです。

機能
ルーティンのチェック — 今日やった項目をタップするだけで記録。1日1回まで。
ポイント管理 — 項目ごとに異なるポイントを設定できます。
連続ボーナス — 同じ項目をN日連続で達成するとボーナスポイントが加算されます。
お小遣い換算 — 月間ポイントに応じたお小遣い金額を自動計算します。
支払い管理 — 月ごとに「支払済み」を記録でき、払い忘れを防ぎます。
履歴カレンダー — 過去の記録をカレンダーで確認できます。
過去の手入力 — 日付を指定して過去の記録を追加できます。
バックアップ — データをJSONファイルでエクスポート・インポートできます。

使い方
このページを開く
「今日」タブでやった項目をタップ
月末に「お小遣い」タブで確認して支払い済みにする

データについて
データはブラウザのlocalStorageに保存されます。端末やブラウザが変わるとデータは引き継がれません。定期的に「設定 → エクスポート」でバックアップを取ることをおすすめします。
